# RAGAM v1.0.0 Release Notes

Welcome to the initial release of **RAGAM**, a modern, feature-rich desktop music streaming application! This release brings a premium listening experience straight to your desktop with a focus on aesthetics, reliable playback, and customization.

## üéâ Key Features

### Beautiful User Interface
- A sleek, modern design built with React, Tailwind CSS, and Radix UI components.
- Responsive layouts that adapt to your window size.

### Multiple Customizable Themes
- Personalize your application with distinct UI themes:
  - **KDON**: A sleek and modern default theme.
  - **MelloStudio**: A soft, elegant theme for relaxed listening.
  - **Cyberpunk**: A vibrant, high-contrast theme available in both Light and Dark modes.

### Advanced Media Playback
- High-quality audio streaming powered by custom HLS proxies and `yt-dlp`.
- Support for intelligent caching: Songs are cached locally during playback to save bandwidth on repeated listens.
- Built-in proxy streaming endpoints to bypass CORS and directly stream audio.

### Offline Library
- Seamless offline support: Save your favorite playlists and individual tracks directly to your local library.
- Listen to your saved music even when an internet connection is unavailable.

### Synchronized Lyrics
- Real-time synchronized lyrics using LRC formats.
- An interactive swipe-up interface built directly into the player to easily follow along with your favorite tracks.

### Multi-Platform Support
- Fully packaged desktop application supporting **Windows**, **macOS**, and **Linux**.
